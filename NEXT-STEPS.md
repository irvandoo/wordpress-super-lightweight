### 4. **Optimasi SEO On-Page** 🔍
**Waktu: 1 jam**

#### Checklist:
- [ ] Install Yoast SEO atau Rank Math (optional)
- [ ] Set permalink structure: `/%postname%/`
- [ ] Submit sitemap ke Google Search Console
- [ ] Tambahkan Google Analytics
- [ ] Set up robots.txt
- [ ] Buat sitemap.xml (WordPress auto-generate)
- [ ] Optimasi meta descriptions
- [ ] Add alt text ke semua gambar

---

### 5. **Performance Optimization** ⚡
**Waktu: 1-2 jam**

#### Checklist:
- [ ] Install caching plugin (WP Super Cache / W3 Total Cache)
- [ ] Compress images (TinyPNG, ShortPixel)
- [ ] Enable Gzip compression
- [ ] Minify CSS/JS (Autoptimize)
- [ ] Lazy load images
- [ ] Use CDN (Cloudflare - gratis)
- [ ] Test dengan GTmetrix
- [ ] Test dengan Google PageSpeed Insights
- [ ] Target: 90+ score
